BYTE ; it is a data type or data definition(.data)


Formula (2^n)-1 = unsigned
Signed minimum range = 2^(n-1)
Signed Maximum Range = (2^(n-1)-1)
where n represents number of bits i.e(n = {8,16,32})

mov syntax:

mov	destination,source
	Register, Register ; example: eax, ebx
	Register, mem ; Eax, val1
	Register, immediate ; Eax, 15
Radix:

To represent the number system we use radix.

h -> hexadecimal
c -> binary
d -> decimal

.data
	;variables

				 ;int val1 = 10

	val2 BYTE ? ; This is a variable declaration
	val3 BYTE 10, 20, 30 ;multiple initialization
